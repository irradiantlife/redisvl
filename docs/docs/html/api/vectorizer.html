OpenAI text vectorizer

This vectorizer uses the OpenAI API to create embeddings for text. It requires an API key to be passed in the api_config dictionary. The API key can be obtained from https://api.openai.com/.

Initialize the OpenAI vectorizer.

Parameters:

  • model (str) – Model to use for embedding.

  • api_config (Optional[Dict], optional) – Dictionary containing the API key. Defaults to None.

Raises:

  • ImportError – If the openai library is not installed.

  • ValueError – If the API key is not provided.

VertexAI text vectorizer

This vectorizer uses the VertexAI Palm 2 embedding model API to create embeddings for text. It requires an active GCP project, location, and application credentials.

Initialize the VertexAI vectorizer.

Parameters:

  • model (str) – Model to use for embedding.

  • api_config (Optional[Dict], optional) – Dictionary containing the API key. Defaults to None.

Raises:

  • ImportError – If the google-cloud-aiplatform library is not installed.

  • ValueError – If the API key is not provided.
